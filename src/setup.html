SetUp

Remember that setting up a project like this involves some technical knowledge, so don't hesitate to seek help from a web developer or IT professional if needed. This guide provides a general overview, and the specific steps might vary based on your hosting provider and server configuration. Get the code

HOW TO DOWNLOAD AND SETUP

Step-1
Download File from Drive

Go to your Google Drive and locate the "SecureFileShare" project folder. Right-click the folder and select "Download". This will download a ZIP file containing the project files.

Step-2
Extract ZIP File

After downloading, extract the ZIP file to a location on your local computer. The extracted folder should contain the project's HTML, CSS, JavaScript, PHP, and other files.

Step-3
Upload Files to Server

Use an FTP client (e.g., FileZilla) or your hosting provider's file manager to upload the extracted project folder to your web server's public directory (often called public_html or www).

Step-4
Create MySQL Database

Log in to your hosting control panel. Create a new MySQL database and note down the database name, username, password, and hostname.

Step-5
Import Database Tables

In your hosting control panel, look for a tool like phpMyAdmin. Use phpMyAdmin to import the database file that came with the project (if provided). This will create the necessary tables.

Step-6
Configure Database

Inside the project folder, locate the database.php file. Open database.php in a text editor and update the database connection parameters (hostname, username, password, database name) with the values from your MySQL database.

HOW TO HOST YOUR PROJECT

Step-1
Sign Up for 000webhost:

Visit the 000webhost website https://www.000webhost.com/ . Sign up for a free account and follow the instructions to set up your account.

Step-2
Create a New Website

After signing up and logging in, click on the "Build website" button. Choose the "Upload Own Website" option.

Step-3
Upload to 000webhost

In the file manager, click on the "Upload" button and select the ZIP file containing the "SecureFileShare" project. Once uploaded, click on the uploaded ZIP file and select "Extract" to extract the files.

Step-4
Create MySQL Database

Go back to the 000webhost dashboard and find the "Manage databases" section. Create a new MySQL database and note down the database name, username, password, and hostname.

Step-5
Configure Connection

Inside the project folder, locate the database.php file. Open database.php in the built-in code editor or a text editor and update the database connection parameters (hostname, username, password, database name) with the values from your 000webhost MySQL database.

Step-6
Access the Project

Go back to the "Website Manager" and click on your website's name. Access the website's URL provided by 000webhost to access the project (e.g., https://yourwebsite.000webhostapp.com).

Step-7
Testing

Test the user registration, login, file upload, and download functionalities thoroughly. If you encounter any issues, refer to the error messages, check your database configuration, and review your server logs for additional information.
