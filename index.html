FileVault

FileVault is a user-controlled file management system designed to provide a secure and seamless experience for uploading, storing, and downloading files. This project prioritizes user security and privacy, offering a platform where users have full control over their files.

FileVault Is

Users enter credentials on the login page, which are authenticated and compared to stored hashed passwords on the server. Successful authentication creates a session, enabling access to authorized resources.

The upload page allows users to easily add files to the system. Users select files from their devices, providing a brief description if needed. Once uploaded, files are securely stored on the server, ensuring data integrity.
